# 更新日志

## [1.0.0] - 2026-06-12

### 新增

- 班级事项发布、编辑、删除功能。
- 班级成员身份认证系统。
- 管理员和超级管理员权限体系。
- 内容安全检测（文本+图片）。
- 订阅消息提醒功能。
- 操作日志记录。
- 首页和事项详情分享功能。

### 安全

- 云函数层权限校验。
- 关键操作日志审计。
- 一次性邀请码机制。

### 改进

- 事项详情页仅通过事项 ID 导航，并按需读取最新数据。
- 事项置顶和取消置顶通过云函数执行权限校验和操作日志记录。
- 订阅授权通过云函数绑定当前用户，消息发送在服务端校验发布权限。
- 发布、编辑和邀请码尝试使用事务计数限制频率。
- 操作日志不再重复保存姓名、学号或事项标题。
